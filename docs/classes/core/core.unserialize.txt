function unserialize($arr)

Takes the serialized form of an object and unserializes it, thus creating a new object that contains the same data as the serialized object. 

<i>Parameters:</i>

- $str
The serialized string. 

- $parent
The id of the folder under what to create the new object. 

- $period - optional 
The period to create the new object under. Defaults to the current period.
