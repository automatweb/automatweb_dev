calls the save method on all the members of the tree

<b>errors:</b>
- throws acl error, if user has no write access to any object in the tree

<b>parameters:</b>
none

<b>returns:</b>
count of objects saved

<b>examples:</b>
#php#
$ot = new object_tree(array(
		"parent" => 90
	));
$ot->foreach_o(array(
		"func" => "set_name",
		"params" => array("automatweb"),
		"save" => false
	));
$ot->save();
#/php#
