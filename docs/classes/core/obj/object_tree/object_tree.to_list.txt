returns an object_list instance that contains all the objects in the current tree

<b>errors:</b>
none

<b>parameters:</b>
none

<b>returns:</b>
object list instance that contains all the objects in the current tree

<b>examples:</b>
#php#
$ot = new object_tree(array(
"parent" => 5,
"name" => "%foo%",
"class" => CL_BAR,
"cool_property" => true
));
$ol = $ot->to_list();
#/php#
