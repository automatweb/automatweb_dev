deletes all objects in the tree and clears the tree

<b>errors:</b>
- acl error is thrown if user has no delete access for any object in the tree

<b>parameters:</b>
none

<b>returns:</b>
the number of objects deleted

<b>examples:</b>
#php#
$ot = new object_tree(array(
		"parent" => 90,
		"name" => "tunafish",
	));
$ot->delete();
#/php#
