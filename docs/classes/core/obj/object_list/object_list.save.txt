calls the save method on all the members of the list

<b>errors:</b>
- throws acl error, if user has no write access to any object in the list

<b>parameters:</b>
none

<b>returns:</b>
count of objects saved

<b>examples:</b>
#php#
$ol = new object_list(array(
		"class" => CL_FILE
));
$ol->foreach_o(array(
		"func" => "set_parent",
		"params" => array(888),
		"save" => false,
	));
$ol->save();
#/php#
