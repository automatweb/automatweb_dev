creates a new object list from an array and returns the new instance

<b>errors:</b>
none

<b>parameters:</b>
- array, key is object id, value is object instance or array to create object
instance from,required
type: array

<b>returns:</b>
instance of object list, with the passed objects inserted in the list

<b>examples:</b>
#php#
$ol = new object_list(array(
		"class" => CL_FILE
	));
$files = $ol->arr();
$ol2 = object_list::from_arr($arr);
#/php#
