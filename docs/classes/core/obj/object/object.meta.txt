returns the value of the specified metadata field

<b>errors:</b>
none

<b>parameters:</b>
- key of the metadata element
type: integer/string

<b>returns:</b>
value of the metadata element, NULL if no element by that name is found, NULL 
if no current object exists

<b>examples:</b>
#php#
$o =& obj(67);
$val = $o->meta("kala");#/php#
