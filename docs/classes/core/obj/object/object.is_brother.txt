returns true or false, depending if the object is a brother object or not

<b>errors:</b>
none

<b>parameters:</b>
none

<b>returns:</b>
boolean value, true if the object is a brother object, false if the object is the original
object, NULL if no object is loaded

$o = obj(89);
if ($o->is_brother())

	$orig = $o->get_original();
