merges the properties passed with the current object properties

<b>errors:</b>
none, if implicit save is off

<b>parameters:</b>
- array of properties to merge, required
type: array, key is property name, value is property value

<b>returns:</b>
none

<b>examples:</b>
#php#
$o = obj(45);
$o->merge_prop(array(
	"foo" => "bar",
	"name" => "sweet zombie jesus"
));#/php#
