returns an instance of the class that is registered to handle this type of object

<b>errors:</b>
- if no class id is set, error is thrown

<b>parameters:</b>
none

<b>returns:</b>
instance of the class that holds the properties for the current object

<b>examples:</b>#php#
$o =& obj(5);
$instance = $o->instance();
$instance->show($o);#/php#
