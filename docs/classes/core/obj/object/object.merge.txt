merges the array given as parameter with the current object's data

<b>errors:</b>
none, if implicit save is off

<b>parameters:</b>
- array of parameters to merge, required
type: array

<b>returns:</b>
none

<b>examples:</b>
#php#
$o =& obj(90);
$o->merge(array(
		"name" => "foo",
		"meta" => array(
			"bar" => 12
		),
		"properties" => array(
			"killer" => "tiger"
		)
));
#/php#
