returns the dirtyness of the object's cache

<b>errors:</b>
none

<b>parameters:</b>
none

<b>returns:</b>
boolean value, true if the object's cache is dirty, false if the cache is valid, 
NULL if no current object exists

<b>examples:</b>#php#
$o = obj(56);
if ($o->is_cache_dirty())
{
	echo "cache is dirty!";
}#/php#
