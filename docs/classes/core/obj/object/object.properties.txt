returns an array of all the properties and their values for the object

<b>errors:</b>
- error is thrown if no object is loaded 

<b>parameters:</b>
none

<b>returns:</b>
array of properties, proprty names are keys, property values are values

<b>examples:</b>#php#
$o = obj(6);
$props = $o->properties();
echo $props["name"];#/php#
