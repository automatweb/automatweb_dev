returns the period id of the period that the object currently has

<b>errors:</b>
none

<b>parameters:</b>
none

<b>returns:</b>
id of the period that the current object has, 0 if no period is set, NULL if no object
is currently loaded

<b>examples:</b>
#php#
$o =& obj(90);
$per = $o->period();
#/php#
