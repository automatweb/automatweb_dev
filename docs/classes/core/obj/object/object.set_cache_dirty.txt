sets the cache expired flag for the current object, this is also set automatically 
if the object is changed

<b>errors:</b>
none, if implicit save is off

<b>parameters:</b>
- cache expired flag, if true, cache is marked as expired, if false, cache is 
marked as valid
type: boolean

<b>returns:</b>
the previous value for the cache dirty flag

<b>examples:</b>#php#
$o =& obj(3);
$o->set_cache_dirty(true);#/php#
