sets all the flags that the current object has

<b>errors:</b>
none, if implicit save is off

<b>parameters:</b>
- flags that contains all the flags, required.
type: integer

<b>returns:</b>
previous state of the flags

<b>examples:</b>
#php#
$o = obj(67);
$fl = $o->flags();
$fl |= FL_FOO;
$fl |= FL_BAR;
$o->set_flags($fl);#/php#
