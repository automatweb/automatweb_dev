sets whether the loaded object will be saved whenever a property is changed or only
if the save() method is called. the default is off.

<b>errors:</b>
none

<b>returns:</b>
the previous state of the implicit save flag

<b>examples:</b>
#php#
$o = obj(666);
$o->set_implicit_save(true);
$o->set_name("foo"); // object is now also automatically saved
#/php#
