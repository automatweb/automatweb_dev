returns the state of the flag that is the parameter

<b>errors:</b>
none

<b>parameters:</b>
- flag id, required
type: integer

<b>returns:</b>
the state of the flag for the current object, NULL if no object is loaded

<b>examples:</b>#php#
$o = obj(78);
if ($o->flag(FL_KALA))
{
		echo "you hafe fish!";
}#/php#
