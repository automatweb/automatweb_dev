returns the object that the connection starts from

<b>errors:</b>
error is thrown, if no current connection exists

<b>parameters:</b>
none

<b>returns:</b>
object instance of the object that the connection starts from

<b>examples:</b>#php#
$conn = new connection(90);
$from = $conn->from();
echo "name = ".$from->name();
#/php#
