returns the object that the connection points to

<b>errors:</b>
error is thrown, if no current connection exists

<b>parameters:</b>
none

<b>returns:</b>
object instance of the object that the connection points to

<b>examples:</b>#php#
$conn = new connection(90);
$to = $conn->to();
echo "name = ".$to->name();
#/php#
