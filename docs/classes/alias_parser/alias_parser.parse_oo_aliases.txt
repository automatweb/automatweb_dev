<b>alias_parser::parse_oo_aliases($oid,&$source,$args = array())</b>

finds aw aliases in the given string and replaces them with their output

<b>errors:</b>
none

<b>parameters, passed by position:</b>
$oid - the object to which the text belongs to. the aliases in the text must be connected to this object, it is used to resolve the aliases to real objects
$source - the text containing the aliases to be parsed. the text is modified so, that aliases are replaced with the output of their objects' parse_alias methods
$args - array of other parameters:
	$templates - gets passed to the parse_alias method of the handler class as the parameter $tpls as a reference

<b>returns:</b>
nothing

Example:

$document = object($document_id);
$content = $document->prop("lead")."&lt;br&gt;".$document->prop("content");

$alp = get_instance("alias_parser");
$alp->parse_oo_aliases($document->id(), $content);

echo $content; // displays document with aliases parsed
