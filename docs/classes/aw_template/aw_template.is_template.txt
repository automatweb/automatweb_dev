<b>aw_template::is_template($name)</b>

checks if a SUB with the gen name exists in the currently loaded template

<b>errors:</b>
none

<b>parameters, passed by position:</b>
name - name of the SUB to check for

<b>returns:</b>
true if the SUB is in the current template, false if no template is loaded or no such SUB exists

Example:

$tpl = new aw_template;

$tpl->read_any_template("foo.tpl");

if ($this->is_template("FOO"))
{
	echo $this->parse("FOO");
}
