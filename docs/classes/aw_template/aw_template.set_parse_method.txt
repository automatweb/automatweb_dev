<b>aw_template::set_parse_method($method)</b>

sets the template parsing method used by the current instance of the template parser. the available methods are:
"" - the default - uses regular expressions to parse the template
"eval" - uses php's eval() function to parse the template - this is faster, but slightly incompatible with the default

<b>errors:</b>
none

<b>parameters, passed by position:</b>
$method - optional, defaults to ""

<b>returns:</b>
none

Example:

$tpl = new aw_template;
$tpl->set_parse_method("eval");
$tpl->read_template("aa.tpl");
echo $tpl->parse(); // this is now slightly faster
