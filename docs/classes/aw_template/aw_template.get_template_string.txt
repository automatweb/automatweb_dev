<b>aw_template::get_template_string($name)</b>

returns the un-parsed content of the given SUB 

<b>errors:</b>
none

<b>parameters, passed by position:</b>
name - name of the SUB whose content to return

<b>returns:</b>
the un-parsed content of the given SUB 

Example:

-- template foo.tpl

<!-- SUB: BOO_FOO -->
{VAR:allah}
<!-- END SUB: BOO_FOO -->

-- code
$tpl = new aw_template;

$tpl->read_any_template("foo.tpl");

$str = $tpl->get_template_string("BOO_FOO"); // returns "{VAR:allah}"
