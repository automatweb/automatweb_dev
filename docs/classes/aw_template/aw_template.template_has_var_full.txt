<b>aw_template::template_has_var_full($varname)</b>

checks if the currently loaded template's contains a VAR with the name given. 

<b>PERFORMANCE WARNING: this is quite slow, do not use often </b>

<b>errors:</b>
none

<b>parameters, passed by position:</b>
varname - name of the VAR to check for

<b>returns:</b>
true if the VAR is in the current template, false if no template is loaded or no such VAR exists

Example:

$tpl = new aw_template;

$tpl->read_any_template("foo.tpl");

if ($this->template_has_var_full("allah"))
{
	$this->vars(array(
		"allah" => "akhbar"
	));
}
