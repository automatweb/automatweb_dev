<b>aw_template::is_in_parents_tpl($tpl, $parent)</b>

checks if the SUB $tpl is a child SUB of the $parent SUB. checks the full chain of SUB's

<b>errors:</b>
none

<b>parameters, passed by position:</b>
tpl - the SUB whose parent SUB's you want to check
parent - the name of the parent SUB to check

<b>returns:</b>
true if the $tpl SUB has a parent SUB by the name $parent

Example:

-- template foo.tpl

<!-- SUB: AHH -->
	<!-- SUB: BAR -->
		<!-- SUB: FOO -->
		<!-- END SUB: FOO -->
	<!-- END SUB: BAR -->
<!-- END SUB: AHH -->

-- code
$tpl = new aw_template;

$tpl->read_any_template("foo.tpl");

$res = $tpl->is_in_parents_tpl("FOO", "AHH"); // returns true
