<b>aw_template::read_adm_template($name, $silent = 0)</b>

reads the given template from the admin template folder


<b>errors:</b>
if file is not found, error is thrown

<b>parameters, passed by position:</b>
name - name of the file to load
silent - optional - is set to 1, no errors are thrown, instead false is returned on error

<b>returns:</b>
false if silent == 1 and file is not found, else true

Example:

$tpl = new aw_template;

$tpl->read_adm_template("foo.tpl");
$tpl->vars(array(
	"foo" => "Mr. T"
));
echo $tpl->parse();
