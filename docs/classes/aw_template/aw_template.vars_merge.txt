<b>aw_template::vars_merge($params)</b>

imports variables into the current template, appending to the previous variables of the same name

<b>errors:</b>
none

<b>parameters, passed by position:</b>
params - array of name => value pairs that are used as variable name => variable value

<b>returns:</b>
none

Example:

-- template foo.tpl

{VAR:allah}

-- code
$tpl = new aw_template;

$tpl->read_any_template("foo.tpl");

$tpl->vars(array(
	"allah" => "allah"
));

$tpl->vars_merge(array(
	"allah" => " akhbar"
));
echo $tpl->parse(); // prints allah akhbar
