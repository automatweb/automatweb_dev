<b>aw_template::use_template($source)</b>

reads the template from the string given


<b>errors:</b>
none

<b>parameters, passed by position:</b>
source - the template content

<b>returns:</b>
true

Example:

$tpl = new aw_template;
$str = "{VAR:foo} is here";

$tpl->us_template($str);
$tpl->vars(array(
	"foo" => "Mr. T"
));
echo $tpl->parse();
