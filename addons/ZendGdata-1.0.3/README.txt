INTRODUCTION
------------

Thank you for downloading Zend Gdata. The files in this archive
are released under the Zend Gdata license.  See LICENSE.txt
in this directory for a copy of the license.  Please see NEWS.txt
in this directory for a high-level list of changes in this release.

New Zend Gdata releases are made available on a regular basis,
when improvements and bugfixes are made. Please visit the website
http://framework.zend.com for more information. To be notified of new
releases, please subscribe to the fw-announce mailing list by sending
a blank message to fw-announce-subscribe@lists.zend.com.


DOCUMENTATION
-------------

Zend Gdata documentation is available on the website:

http://framework.zend.com/manual


ZEND GDATA CLASSES
----------------------

Zend Gdata classes can be found in the following directory:

/library

Your PHP include_path should contain the Zend Gdata library. For
more information, see INSTALL.txt.
